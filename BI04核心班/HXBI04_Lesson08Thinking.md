# Q1:今天讲解了时间序列预测的两种方式，实际上在数据库内建时间属性后，可以产生时序数据库，请思考什么是时序数据库？为什么时间序列数据成为增长最快的数据类型之一?

## A1: 

1. 什么是时序数据库？

+ 时序数据是按照时间顺序组成的数字序列，这类数据是在不同时间上收集到的数据，用于所描述现象随时间变化的情况。 这类数据反映了某一事物、现象等随时间的变化状态或程度。
+ 时序数据库就是存放时序数据的数据库，支持快速写入、持久化、多维度的综合查询等基本功能，如：InfluxDB数据库。

2. 为什么时序数据称为增长最快的数据类型之一？

+ 除了传统手机、计算机在实时采集数据之外，手环、共享单车、出租车、智能手表、环境监测设备、电梯、大型设备、工业生产线等等在源源不断的产生海量的实时数据并发往云端。
+ 这些所有机器、设备、传感器以及交易系统都会产生时序数据，典型特点是：
  + 产生频率快（每一个监测点一秒钟内可产生多条数据）；
  + 严重依赖于采集时间（每一条数据均要求对应唯一的时间）；
  + 测点多信息量大（常规的实时监测系统均有成千上万的监测点，监测点每秒钟都产生数据，每天产生几十GB的数据量）。
+ 随着物联网、大数据和人工智能技术的发展，在数据随着时间而增长的过程中，时序数据成为增长最快的数据类型之一。

# Q2:BCG Matrix（波士顿矩阵）四象限分别代表什么？不同象限，有怎样的数据决策?

## A2:

+ 波士顿矩阵（BCG Matrix），又称市场增长率-相对市场份额矩阵，以相对市场占有率(以下简称:占有率)为横轴、相对市场增长率(以下简称:增长率)为纵轴，将企业产品的具体情况分为四个象限：

  + 明星产品：处于高增长率、高占有率的产品；这类产品一般处于产品生命周期的成长期，应加大投资、提高占有率，增强其竞争地位；未来可能成为今年产品；
  + 金牛产品：处于低增长率、高占有率的产品；说明产品进入成熟期，销量大，可以为企业提供资金，对这类产品可以减少投资，维持现存市场或延缓其占有率下降速度即可；
  + 问题产品：高增长率、低占有率的产品，新产品的表现往往如此；高增长率，说明市场机会大、前景好，低占有率说明营销不足，对于符合公司发展目标的问题产品，需要增加投资，改进营销策略，拓展占有率；对于不符合公司发展目标的问题产品，需要舍弃；
  + 瘦狗产品：低增长率、低占有率的产品，处于衰退期的词频，这类产品占有公司资源，却无法带来收益，应逐渐舍弃，将资源转移到问题产品的发展上去。
